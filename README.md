# College Event Management System

A full-stack web application developed for managing college events efficiently.  
This platform allows students to explore upcoming events, register for participation, and receive event notifications through email.  
Admins can manage events through a dedicated dashboard.

---

## Features

### Student Side
- User Signup and Login
- Explore Upcoming Events
- View Detailed Event Information
- Register for Events
- Participate in activities like dance, singing, etc.
- View College Event Memories Gallery

### Admin Side
- Add New Events
- Update or Remove Events
- Manage Event Information
- Send Email Notifications to Registered Users

---

## Project Workflow

1. Admin creates and manages events
2. Students explore available events
3. Students register through forms
4. Registration data is stored in MySQL database
5. Admin sends email notifications to users
